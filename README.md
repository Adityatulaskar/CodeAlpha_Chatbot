# 🤖 CodeAlpha Chatbot

## 📌 Project Overview

CodeAlpha Chatbot is an interactive AI-powered chatbot developed as part of the CodeAlpha Internship Program. The chatbot is designed to simulate human-like conversations, respond to user queries, and provide an engaging user experience through a simple and intuitive interface.

## 🚀 Installation

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/CodeAlpha_Chatbot.git
```

### 2. Navigate to the Project Directory

```bash
cd CodeAlpha_Chatbot
```

### 3. Install Dependencies

```bash
pip install -r requirements.txt
```

### 4. Run the Application

```bash
python chatbot.py
```

## 💡 How It Works

1. User enters a message.
2. Chatbot processes the input.
3. The system analyzes the query.
4. An appropriate response is generated.
5. The response is displayed to the user.
